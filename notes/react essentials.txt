ğŸ“ŒTwo way binding in React

In React, two-way data binding means that:
The UI reflects the state, and
The state updates when the UI changes.

In simpler terms:
ğŸ‘‰ When the input field changes, the state is updated,
ğŸ‘‰ And when the state changes, the input field updates.

import { useState } from 'react';

function TwoWayBindingExample() {
  const [name, setName] = useState('');

  return (
    <div>
      <input
        type="text"
        value={name}              // ğŸ‘ˆ Bound to state
        onChange={(e) => setName(e.target.value)}  // ğŸ‘ˆ Updates state on input change
      />
      <p>Hello, {name}!</p>       {/* ğŸ‘ˆ State reflects in UI */}
    </div>
  );
}

-------------------------------------------ğŸ§±Immutable Update Game Board ----------------------------
ğŸ“ŒImmutable Update
To update a specific cell (e.g. row 1, col 2) in a React state object (2D array) without mutating the original state.

So, instead of doing this âŒ:
prevGameBoard[rowIndex][colIndex] = 'X'; // âŒ MUTATES the original array

We do this âœ…:
âœ…Immutable Update (Line-by-line):
setGameBoard((prevGameBoard) => {
  const updatedBoard = [...prevGameBoard.map(innerArray => [...innerArray])];
  updatedBoard[rowIndex][colIndex] = 'X';
  return updatedBoard;
});

What it does:
1.prevGameBoard.map(innerArray => [...innerArray])
  â†’ Creates a deep copy of the 2D array.
  â†’ Each row is copied ([...innerArray]) so you don't accidentally mutate it.

2.updatedBoard[rowIndex][colIndex] = 'X'
  â†’ Set the chosen square to 'X'.

3.return updatedBoard
  â†’ Set the new board as state using setGameBoard.

ğŸ–¼ Output:
Each button in the 3x3 grid will show 'X' when clicked, and React will re-render correctly.
